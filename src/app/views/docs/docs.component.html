Introduction

Welcome to the JDM API, a platform designed for Japanese car enthusiasts. Here you will find detailed information on various models, technical specifications and key features.

This guide will help you understand how to use the API effectively and get the most out of its features.

Main Route

The API starts from the main path provided by Render: https://jdm-api.onrender.com/v1/cars. From this basis, you can access all available endpoints.

Pagination

The API uses a paging system to facilitate navigation between large volumes of data. You can use the parameters ?page and ?limit to monitor the results

Example of input:

GET https://api-jdm.onrender.com/v1/cars?page=1&limit=2

Structure and Endpoints

The API is organised in clear and simple paths, starting with the base /v1/cars. From here, you can access various endpoints to obtain car lists, specific details, statistics and perform advanced searches according to your needs.

  • /v1/cars o /v1/cars/: Gets all stored cars (getAllCars).
  • /v1/cars/:id: Gets the detailed information of a specific car via its car ID (getOneCar).
  • /v1/cars/random: Returns a random car (getRandomCar).
  • /v1/cars/brands: Displays all available brands (getAllBrands).
  • /v1/cars/stats: Displays the average of all car statistics (getGlobalStat).

Basic example of use: https://api-jdm.onrender.com/v1/cars?brand=Mazda to filter by brand.

Parameters and Filtering

The API allows you to use multiple parameters to filter the results and find specific models easily. You can combine multiple filters on the same URL to refine your searches according to your needs. Some examples are shown below:

  • /v1/cars?brand=Nissan: Filter by brand.
  • /v1/cars?model=Supra: Filter by model.
  • /v1/cars?horsepower=276: Filter by horsepower.
  • /v1/cars?engine=2JZ-GTE: Filter by engine type.
  • /v1/cars?country=Japan: Filter by country of origin.
  • /v1/cars?transmission=Manual: Filter by transmission type.
  • /v1/cars?drivetrain=RWD: Filter by type of traction.
  • /v1/cars?max_speed=250: Filter by maximum speed.
  • /v1/cars?acceleration=5.0: Filter by acceleration.
  • /v1/cars?weight=1400: Filter by weight.
  • /v1/cars?price=25000: Filter by price.
  • /v1/cars?fuel_consumption=10.0%20L/100%20km: Filter by fuel consumption.
  • /v1/cars?yearStart=1995&yearEnd=2000: Filter by year range.

In addition, it is possible to combine multiple parameters to obtain more accurate results. For example:

https://api-jdm.onrender.com/v1/cars?brand=Subaru&engine=EJ20%20Turbo&transmission=Manual
